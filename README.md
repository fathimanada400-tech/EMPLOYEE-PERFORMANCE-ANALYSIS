📊 Employee Performance Analysis Dashboard

This project showcases an interactive Employee Performance Analysis Dashboard designed to provide insights into workforce trends and key HR metrics. The dashboard visually summarizes employee data to support better decision-making.

🔍 Key Features:

KPIs Overview
Total Work Hours
Work-Life Balance Score
Monthly Income Aggregation
Education Analysis
Distribution of employees across different education fields (e.g., Life Sciences, Medical, Marketing)
Department Filter

Dynamic filtering by departments:

Human Resources
Research & Development
Sales
Salary Insights
Monthly rate distribution across job roles
Workload vs Standard Hours
Comparison of total daily rates and standard working hours
Attrition Analysis
Employee attrition trends categorized by age groups
Job Level Distribution
Department-wise comparison of job levels

🛠️ Tools & Technologies:

Data Visualization Tool (e.g., Power BI / Tableau / Excel)
Interactive filters and charts
Data modeling and aggregation techniques
📈 Purpose:

This dashboard helps organizations:

Understand employee demographics
Monitor attrition patterns
Analyze compensation and workload distribution
Make data-driven HR decisions# EMPLOYEE-PERFORMANCE-ANALYSIS
